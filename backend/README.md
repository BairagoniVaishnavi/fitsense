# 🏋️ Fitsense Backend

## 🚀 Overview
A fitness tracking backend built with Node.js, Express, and MongoDB.

## 🔥 Features
- JWT Authentication
- Workout Tracking
- Streak System
- AI Workout Suggestion
- Analytics Dashboard
- Badge System

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT

## 📡 API Endpoints

### Auth
- POST /api/auth/register
- POST /api/auth/login

### Workouts
- POST /api/workouts
- GET /api/workouts

### Suggestion
- POST /api/suggestion

### Analytics
- GET /api/analytics
